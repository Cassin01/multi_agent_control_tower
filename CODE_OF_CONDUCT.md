# Code of Conduct

## Our Commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus discussion on technical outcomes.
- Assume good intent and ask clarifying questions.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Trolling, personal attacks, or intimidation.
- Publishing private information without consent.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, wiki edits, issues, and contributions that are not aligned with this Code of Conduct.

## Reporting

Report incidents by opening a private security report or contacting the maintainer through GitHub.

This project follows the spirit of the Contributor Covenant.
